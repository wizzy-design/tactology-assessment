# TGlobal Roster Assessment

A roster management interface built with Next.js, TypeScript, and Chakra UI.

## Live Demo

[Vercel URL](https://wisdom-ochei-tactology-assessment.vercel.app/)

## Features

- Drag and drop shift scheduling via @dnd-kit
- Data persistence using localStorage
- Interactive 30-minute interval grid with overlap handling
- Live/Planner view toggling
- Responsive sidebar with employee roster

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Chakra UI v3
- iconsax-reactjs & react-icons

## Getting Started

1. **Install dependencies**:

   ```bash
   npm install
   ```

2. **Run dev server**:

   ```bash
   npm run dev
   ```

3. **Build**:
   ```bash
   npm run build
   npm start
   ```
